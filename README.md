# RobotX

RobotX is a high-performance public blockchain powered by [Nethermind](https://www.nethermind.io/) using Proof-of-Authority (Aura) consensus.

## RX Token

| Property | Value |
|---|---|
| Token Symbol | RX |
| Maximum Supply | 2,100,000,000 (2.1 Billion) |

## Chain Features

- **Consensus**: Nethermind PoA (Aura)
- **Block Time**: 1 second
- **Max Gas per Transaction**: 1,000,000,000 (1 Billion)
- **EVM Upgrades**: Supports [Pectra](https://eips.ethereum.org/EIPS/eip-7600) fork and [Fusaka](https://eips.ethereum.org/EIPS/eip-7607) upgrade features
- **Performance**: Designed for high-throughput, low-latency applications
